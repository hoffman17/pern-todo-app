<h1>PERN Stack Todo App</h1>
<p>This full stack app utilizies ReactJS and Bootstrap to facilitate the User Interface presentation. On the backend, I'm using NodeJS and ExpressJS to create RESTful API endpoints that allow the user to dynamically get, put, post, or delete records from a PostgreSQL Database based on interactions using state from the front end.</p>
